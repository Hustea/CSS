CSS

  1. ¿Qué es CSS y para qué sirve?

    CSS (Cascading Style Sheets) es un lenguaje de diseño utilizado para dar estilo a las páginas web. Define cómo se verán los elementos HTML, como el color, tamaño de fuente y disposición.

  2. Formas de aplicar CSS en HTML

    Hay tres formas principales: en línea (dentro del elemento HTML), interno (dentro de la sección `` del HTML) y externo (en un archivo CSS vinculado).
